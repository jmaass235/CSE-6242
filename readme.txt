#################
###Description###
#################
This project is a Regime-Aware Market Conditions Analyzer that uses a Hidden Markov Model (HMM) to identify macroeconomic market regimes over time.
The model is trained on historical FRED data and classifies each time period into one of three regimes: Low-Volatility Growth, High-Volatility Crisis, and Transition/Tightening.

The backend model processes macroeconomic indicators such as S&P 500 returns, unemployment rate, yield spread, and inflation changes.
The model then outputs a JSON file containing regime classifications, transition probablilities, and summary statistics.

The frontend dashboard (Dashboard) is built using D3.js and visualizes:
Regime Timelines with Cumulative Returns
Macroeconomic Conditions Across Time
Summary Statistics for each regime

Together, the system provides both analytical modeling and an interactive visualization layer for understganding market conditions.


#################
##Installation###
#################

1. Download Python 3.7 is installed
2. Install Required Python libraries:
	hmmlearn
	scikit-learn
	pandas
	numpy
3. Ensure the following files are in the same directory:
	hmm_model.py
	FRED_DATA.csv
	regime_output.json (generated by hmm_model.py)
	Dashboard.html
	

################
###Execution####
################

1. Run the model (hmm_model.py) to generate regime_output.json
2. Start a local server:
	python -m http.server 8000
3. Open a browser and navigate to:
	http://localhost:8000/
4. Open the Dashboard.html file in the browser.
